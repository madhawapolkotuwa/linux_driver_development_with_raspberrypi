
## [File Operations (open, release, read, write)](/07_file_cdev_fops/01_file_ops/)
### Video :
[![Youtube Video](https://img.youtube.com/vi/zDhz6XhUZ9A/0.jpg)](https://www.youtube.com/watch?v=zDhz6XhUZ9A)



* write to device file:
```
echo "hello world" | sudo tee /dev/my_cdev0
```

* read from device file:
```
sudo cat /dev/my_cdev0
```
